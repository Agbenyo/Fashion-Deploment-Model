# Usage Examples

Once you have the application running, you can access it via `http://127.0.0.1:8000`.

## API Endpoints

### 1. Root Endpoint

- **GET** `/`
- Returns a welcome message.

### 2. Train Model

- **POST** `/train`
- Trains the CNN model on the specified dataset.

### 3. Predict

- **POST** `/predict`
- Provides predictions based on input images.
